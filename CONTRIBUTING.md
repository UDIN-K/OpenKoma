## 🚀 Getting Started

1. **Fork** the repository on GitHub.
2. **Clone** your fork locally:
   ```bash
   git clone https://github.com/<your-username>/OpenKoma
   cd OpenKoma
   ```
3. **Install** dependencies:
   ```bash
   npm install
   ```
4. **Start** the development server:
   ```bash
   npm run dev
   ```

## 🛠️ How to Contribute

1. Create a new branch for your changes:
   ```bash
   git checkout -b feature/your-feature-name
   ```
2. Make your changes and commit them with a clear message.
3. Push your branch and open a **Pull Request** against the `main` branch.
4. Describe what your PR does and reference any related issues.

## 📱 Building for Android

If your changes affect the Android build, please test the APK as well. See the full [**Android Build Guide**](./HOW-TO-BUILD-ANDROID.md) for step-by-step instructions.
